Selection Logic

Match products by site conditions, not by generic model names.

For fast deployment

Start with Mobile Solar Power Container when mobility, repeat use, and quick site setup matter most.

For multi-energy input

Consider Mobile Hybrid Power Trailer when wind, PV, storage, diesel, and grid input need to work together.

For parking and charging

Consider Modular Hybrid Solar Carport for parking, service areas, community charging, and infrastructure power.

For system projects

Use Off-grid System Series or Hybrid Power SYS Series when the requirement is a wider project solution.

RFQ Information

Send the site condition first. The product configuration follows the project.

Recommended RFQ information: load demand, operating hours, site location, grid condition, backup power requirement, mobility requirement, and expected project scenario.
